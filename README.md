# LoyaltyLogin

`LoyaltyLogin` គឺជា Plugin សម្រាប់ Minecraft Server ដែលមានមុខងារការពារសុវត្ថិភាពខ្ពស់ សម្រាប់អ្នកលេងដែលប្រើប្រាស់ Cracked Account។

## លក្ខណៈពិសេស (Features)
* **Security First:** ប្រើប្រាស់ `BCrypt` hashing ដើម្បីការពារ Password របស់អ្នកលេងឱ្យមានសុវត្ថិភាពបំផុត។
* **Lockdown System:** រាល់ពេលអ្នកលេងចូល Server ភ្លាម ពួកគេនឹងមិនអាចដើរ, និយាយ (Chat), ឬប៉ះពាល់ Item បានឡើយ រហូតទាល់តែពួកគេ Login បានជោគជ័យ។
* **Commands:**
    * `/register <password>` - សម្រាប់ចុះឈ្មោះលេខកូដសម្ងាត់ថ្មី។
    * `/login <password>` - សម្រាប់ចូលប្រើប្រាស់អាខោន។

## របៀបដំឡើង (Installation)
1. ទាញយក Plugin នេះ (Jar file)។
2. ដាក់ចូលទៅក្នុង Folder `plugins/` របស់ Server បង។
3. Restart Server របស់អ្នក។

## បច្ចេកវិទ្យាដែលប្រើប្រាស់ (Tech Stack)
* **Language:** Java
* **Platform:** Spigot/Paper API (1.21)
* **Security:** jBCrypt (for password hashing)
